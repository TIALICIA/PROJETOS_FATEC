### Conteúdo da Prova

A avaliação contemplará os conteúdos trabalhados desde o início da disciplina até a **Aula 08**, incluindo os conceitos aplicados no **Projeto ABP**.

**Conteúdos abordados:**

* Introdução aos Bancos de Dados Não Relacionais;
* MongoDB: conceitos fundamentais;
* Modelagem de Dados em MongoDB;
* CRUD (Create, Read, Update e Delete);
* Operadores de consulta;
* Paginação (skip e limit);
* Atualizações e exclusões de documentos;
* Embedding e Referencing;
* Aggregation Framework ($match, $group, $sort, $project, $limit e $skip);
* Integração MongoDB com aplicações;
* Monitoramento, Backup e Segurança;
* Aplicação prática dos conceitos no Projeto ABP.
